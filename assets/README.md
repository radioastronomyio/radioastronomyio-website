## Style Notes

All banners follow the Nano Banana 3 style:

- **Background:** Graph paper/blueprint (cream with light grid)
- **Primary Color:** Steel blue (#5B8FAE range)
- **Accent Color:** Coral/salmon (#E8A87C)
- **Style:** Illustrated infographics, flow diagrams

**Exception:** `proxmox-astronomy-lab-repo-banner.jpg` uses richer space imagery treatment as the flagship infrastructure repository.

## Image Requirements

- **Hero images:** ~1200×400px, optimized for web
- **Repo banners:** ~1200×300px
- **Logo:** 200×200px
